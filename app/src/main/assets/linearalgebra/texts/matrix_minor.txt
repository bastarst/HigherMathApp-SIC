**Минор**
**Минором** элемента aᵢⱼ квадратной матрицы называется определитель матрицы, которая получается из исходной путём **удаления i-й строки и j-го столбца**.
Обозначается как: Mᵢⱼ
**Пример:**
Пусть дана матрица:

Найдём минор M₁₂ — это минор элемента из 1-й строки и 2-го столбца (то есть числа 2). Удаляем 1-ю строку и 2-й столбец:
[image, 300, 75] minor12.svg
**Алгебраическое дополнение**
**Алгебраическим дополнением** Aᵢⱼ элемента aᵢⱼ называется минор с учётом знака:
Aᵢⱼ = (-1)ⁱ⁺ʲ • Mᵢⱼ
То есть знак зависит от позиции элемента:
 • если i + j чётное → +
 • если нечётное → -
**Пример:**
Из предыдущего примера:
A₁₂ = (-1)¹⁺² • M₁₂ = (-1)³ • (-5) = -1 • (-5) = 5
**Зачем нужны миноры?**
 • Для вычисления **определителя больших матриц** (разложение по строке/столбцу).
 • Для нахождения **обратной матрицы**.
 • Для **метода Крамера** при решении систем линейных уравнений.
