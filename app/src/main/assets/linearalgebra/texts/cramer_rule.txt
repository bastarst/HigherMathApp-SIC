**Что такое СЛАУ?**
**СЛАУ** — это **система линейных алгебраических уравнений**, состоящая из нескольких линейных уравнений с несколькими переменными.
Пример системы из трёх уравнений с тремя переменными:
[image, 200, 75] slau.svg
**Общая форма в матричном виде**
СЛАУ можно записать в виде матричного уравнения:
A • X = B
Где:
 • A — матрица коэффициентов:
[image, 200, 100] matrix_of_coefficients.svg
 • X — вектор переменных:
[image, 100, 100] vector_of_variables.svg
 • B — вектор свободных членов:
[image, 100, 100] free_members_vector.svg
**Метод Крамера**
Метод Крамера позволяет найти решение СЛАУ, если:
 • Система **квадратная** (число уравнений = числу переменных),
 • Определитель матрицы коэффициентов det(A) ≠ 0

 • Aᵢ — матрица, полученная заменой i-го столбца матрицы A на столбец B
 • det(A) — определитель исходной матрицы

**Когда метод Крамера применять нельзя?**
 • Если det(A) = 0, метод Крамера не работает. Система может не иметь решений или иметь бесконечно много решений.
 • Метод неэффективен для больших систем (больше 3–4 уравнений) из-за большого числа вычислений.
