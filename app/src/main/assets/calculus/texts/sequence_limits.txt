**Что такое последовательность?**
**Последовательность** — это упорядоченный набор чисел, записанный в виде:
[formula, other] a₁, a₂, a₃, ..., aₙ, ...
Каждое число aₙ называется **членом последовательности**, а n — его номером.

**Предел последовательности**
Последовательность **{aₙ}** имеет **предел A**, если при увеличении $n$ члены последовательности всё ближе и ближе приближаются к числу **A**. Обозначается:
[formula, lim] n, ∞, aₙ [formula, other] = [formula, other] A
Говорят: **«предел последовательности aₙ при n → ∞ равен A»**.

**Формальное определение (по Коши)**
Последовательность **{aₙ}** стремится к пределу **A**, если:
Для любого **ε > 0** существует такое **N**, что для всех **n > N**:
[formula, other] |aₙ - A| < ε
То есть, начиная с некоторого номера, все члены последовательности находятся в ε-окрестности точки **A**.

**Примеры**
**Пример 1:**
[formula, fraction] aₙ, 1, n
[formula, lim] n, ∞, 1, n [formula, other] = [formula, other] 0
Чем больше **n**, тем меньше 1/n, и она стремится к нулю.

**Пример 2:**
[formula, other] aₙ = (-1)ⁿ
Последовательность чередуется: **-1, 1, -1, 1, ...**
Предел **не существует**, так как нет одного значения, к которому стремятся члены.

**Свойства пределов последовательностей**
Пусть **lim aₙ = A**, **lim bₙ = B**. Тогда:
 • **lim (aₙ +/- bₙ) = A +/- B**
 • **lim (c  •  aₙ) = c • A для любого числа c**
 • **lim (aₙ  •  bₙ) = A •  B**
 • **lim (aₙ / bₙ) = A / B, если B не равен 0**
